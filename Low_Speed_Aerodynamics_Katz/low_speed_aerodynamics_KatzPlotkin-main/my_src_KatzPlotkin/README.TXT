This is source code (Fortran 77) with some fixes of the programs listed in the Appendix of the book Low Speed Aerodynamics by Katz and Plotkin.

Inside each directory you will find a README.TXT file with some basic instructions on how to compile the source code and how to run the program.

To compile the source you need a fortran compiler.

For plotting we recommend GNUPLOT
