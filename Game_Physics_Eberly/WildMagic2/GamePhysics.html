Game Physics

is another book in the Morgan Kaufmann Series in Interactive 3D Technology. Please see the Table of Contents for more details. For a larger image of the book cover, see book cover (21K bytes). As of July 2003, the book is in the production phase of publication. According to the Morgan Kaufmann Publishers web site, the publication date will be December 15, 2003.

The CD-ROM contains C++ source code for general physics situations and has many examples and applications. The sample applications run on top of the graphics engine, Wild Magic version 2, whose source code will also be included on the CD-ROM. The binding between the Wild Magic data structures and the physics source code is minimal, thus making it easy to modify for your own needs.
