MindGames from Media Lab Europe

MindGames is a research group in Media Lab Europe (based in Dublin, Ireland), the European research partner of the MIT Media Lab. The goal of the group is to develop novel tools and technologies which will allow people to learn beneficial new skills in a virtual environment, in such a way that those skills are retained in, and can be applied to, “real-world” situations. In particular, the group seeks to harness the delight and pleasure that children take in video gaming, to produce a powerful new therapeutic medium for a variety of psychological disorders.

The group has developed a number of games using the Wild Magic platform, including "Relax To Win", "Brain-Child" and "Mental Leaps". Visit the MindGames website to find out more about the group, and its research activities.

Synthetic Characters project at the MIT Media Lab

A group of folks at the MIT Media Lab have an ongoing project for developing Synthetic Characters that exhibit intelligent behavior. The group is headed by Professor Bruce Blumberg. A demonstration called Alpha Wolf was presented at SIGGRAPH 2001 in the Emerging Technologies section. The graphics foundation is a modified version of the Wild Magic engine. The demonstration is a fine example of using graphics to support the main goal, showing the intelligent behavior, without drawing attention away from that goal by gratuitous use of flashy special effects. 3D Studio Max Exporter (Version 1.00)

The latest version by Daragh McDonnell of the MIT Media Lab Europe with support for more features and some bug fixes. This compiles and runs with Wild Magic version 1.3, but does not yet have the support for the new high-level rendering features (exporter updated independently of my WM development). Both a binary (*.dle) and source are available. If you are now using Visual C++ 7.0, please read the ReadMe.txt file before compiling the exporter source code. The zip files expand to MagicSoftware/WildMagic/Tools/MaxToMgc. 3D Studio Max is a 3D modeling package from Discreet.

trueSpace Exporter (Version 1.0.0)

A trueSpace plugin exporter is provided by Sashan Govender. The zip file contains the plugin and a brief description. Sample data may also be downloaded. trueSpace is a 3D design tool from Caligari Corporation.

2MGC X File Converter (Version 1.4)

A standalone 3d file conversion utility provided by Anthony Matarazzo. Converts 3DS and X geometry files. Texture formats: BMP, PCX, PNG, PBM, PPM, PGM, TGA, TIFF, JPEG, GIF, IFF, ILBM, YUV, and more. It will also resize textures to power of two using high quality sizing methods.
Source (requires build setup) or Executable.

Model Viewer and LightWave File Loader (Version 1.0)

The Wild Magic source code distribution now has a new tool written by Dario Accornero. The project is MagicSoftware/WildMagic/Tools/ModelViewer. The program can load *.mgc or *.lwo files and display them. Various keyboard and mouse options allow you to navigate and tumble the model.

WM Scene Viewer (Version 1.0.0)

A scene viewer that combines the rendered scene and a tree control for browsing the scene graph is provided by Sashan Govender. The zip file contains the executable, a sample scene graph and textures, and a brief description. The source code uses the Microsoft Foundation Classes.
