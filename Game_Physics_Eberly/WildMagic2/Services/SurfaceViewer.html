Level surface viewer.

The viewer launches three windows for displaying triangle mesh surfaces. One window is a dialog that allows you to load multiple meshes. The color of each mesh is user controlled. A slider control lets the user control the level of detail of the mesh. The decimation is based on the Garland-Heckbert algorithm.



Another dialog allows the user to control eight point-light sources for illuminating the level surfaces.



The final window displays the meshes. The four images below shows level surfaces for a 3D x-ray crystallography image of the superoxide dismutase molecule. The number above the image name denotes the percentage of number of triangles retained in the decimation.
