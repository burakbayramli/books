This page contains links to source code for various applications and test programs. Some of these are sophisticated while others are designed just to test parts of the library. The files are for browsing. To build and run the applications, you will need to download all the source and build the main libraries for the source code at this site. The 3D applications are built on top of the Wild Magic graphics library and the renderer libraries.
WmlApplication.h , WmlApplication.inl , WmlApplication.cpp , WmlAglApplication.cpp , WmlDxApplication.cpp , WmlGlutApplication.cpp , WmlWglApplication.cpp

The 2D applications are built on top of the 3D application layer! This approach is a relatively simple way to achieve portability across platforms. The current 2D application layer is minimal, but will be modified with new features as needed.
WmlApplication2.h , WmlApplication2.cpp

For applications requiring command line parameter parsing, the system source code folder already has support and is included in the Wild Magic base library. The parsing may be used in windowed as well as console-based applications.
WmlCommand.h , WmlCommand.cpp , Command Line Parsing
