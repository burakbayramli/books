Bead Sliding Down a Spiral Curve:
BeadSlide.cpp , PhysicsModule.h , PhysicsModule.inl , PhysicsModule.cpp

A bead is placed at a location on a spiral curve. The bead is subject only to gravitational force. This program computes the position of the bead for various times. The application is a console one. For an exercise, write a 3D graphics program to display the moving bead. The simulation results are simulation.txt
