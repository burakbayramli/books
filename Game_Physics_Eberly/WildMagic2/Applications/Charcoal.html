Shading with a Charcoal Appearance:
Charcoal.h , Charcoal.cpp , Background.cg , Background.cg

This is an implementation of Charcoal-style NPR rendering written by Michael Riegger for cgshaders.org. The implementation is based upon the following paper: 
  Aditi Majumder (University of North Carolina)
  M. Gopi (University of California, Irvine)
  "Hardware Accelerated Real Time Charcoal Rendering"
This shader was modified slightly for use in WildMagic.
