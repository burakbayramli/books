The program displays the biped with an environment map.  The specular
lighting is drawn so that the environment texture does not tint it.  Press
the 'L' key to turn off the effect.  The specular effect is still there, but
the specular highlights are tinted by the texture.  The 'L' key is a toggle
to turn on/off the effect.

Keyboard controls.
l/L : toggle the specular-after-texture effect
q, Q, ESC:  quit the application
