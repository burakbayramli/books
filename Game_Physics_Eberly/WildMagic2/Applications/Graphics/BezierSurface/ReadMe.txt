Command line usage.
1.  To display the triangle patch, use "BezierSurface -t"
2.  To display the rectangle patch, use "BezierSurface -r"
3.  To display the cylinder patch, use "BezierSurface -c"
The default without options is the triangle patch.

Camera controls.
Up/Down arrow:     translate along camera direction vector
Left/Right arrow:  rotate about camera up vector (yaw)
Home/End:          translate along camera up vector
PageUp/PageDown:   rotate about camera left vector (pitch)

Keyboard controls.
q, Q, ESC:  quit the application
0:          reset frame rate counter
w,W:        toggle wireframe
+,-:        increase/decrease tessellation level
<,>:        increase/decrease second tessellation level (cylinder only)
