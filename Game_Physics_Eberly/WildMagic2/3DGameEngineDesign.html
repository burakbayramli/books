3D Game Engine Design

by David H. Eberly, is the first book in the Morgan Kaufmann Series in Interactive 3D Technology. The book ships with the Wild Magic real-time 3D graphics source code. As of October 21, 2003, the source version is 2.11. The current platforms that are supported by Wild Magic are Microsoft Windows 9x/NT/2000/XP, Linux, and Macintosh OS X.

Visit the Amazon.com page for this book to purchase it.

General Information. The book is 561 pages long. It includes a CDROM with source code, but you will want to visit the source code page to download the latest version. The book has had three printings, the latest containing version 0.4 of the software. The current version downloadable version is 2.11. Various book samples are available as PDF files. These files launch in a separate window. The table of contents (218K) gives a brief idea of what topics are covered in the book. Other samples from the book include the preface (98K), a list of figures (139K), a list of tables (121K), the introduction chapter (216K), and the chapter Hierarchical Scene Representations: pages 141 to 250 (325K), pages 151 to 167 (300K). The book corrections list the known errors in the book itself. The changed pages for the third printing are in the same font and style as the book, so you can print pages from this document and paste them on top of the pages in your book copy. If you find what you believe to be errors in the text, please send an email to Support with a description of the problem. For a larger image of the book cover, see book cover (56K bytes).

Versions of the Book, Software, and Software Updates. The book is in its third printing. The CDROM that shipped with the first printing is Version 0.1. The CDROM that shipped with the second printing is Version 0.2. The CDROM that shipped with the third printing is Version 0.4. The top level directory of the CDROM clearly indicates which version you have. Now available at the web site is Wild Magic Version 2.11 and includes distributions for Windows 9x/NT/2000/XP, Linux, and MacIntosh. You can download the latest source from the source code page. If you discover bugs or are having problems with the software, please let me know by sending an email to Support with a description of the problem. See the Update History for bug fixes and other relevant changes to the libraries.
