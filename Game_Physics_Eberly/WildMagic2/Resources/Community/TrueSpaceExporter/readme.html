trueSpace To Magic Version 1.0.0

Introduction

This is a trueSpace 5 plugin that will export the scene to *.mgc files usable by the WildMagic 3D Game Engine. It only converts geometry, material and texture information. No animation information is converted.

Usage:

Copy the file WMExporter.tsx into the following directory <truespace folder>\plugins\WMExporter. Start trueSpace and install the plugin like you would any other trueSpace plugin.
