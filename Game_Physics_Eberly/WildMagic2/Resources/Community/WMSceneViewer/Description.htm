WMSceneViewer

Introduction:

This is a program that lets you view the scene as a tree and in a 3D view. It allows you change some variables and properties of the scene graph through the tree view. For example you can change the color of the lights, add another texture stage or change the blending operations. It supports cutting and pasting of nodes. You can also save changes made to the scene. You can rotate around the scene by using the arrow keys.

Usage:

What follows describes how to add another texture stage to the scene. The files mentioned below are in the zip file.
  • Open the file sphere_cloud.mgc.
  • Expand the items in the tree view until you find the TextureState object.
  • Go to the edit menu and choose Add Texture.
  • Choose the file cloud_gray.mif.
  • Now if you double-click on one of the properties of the Texture, like the Apply, you can change the Apply mode.
