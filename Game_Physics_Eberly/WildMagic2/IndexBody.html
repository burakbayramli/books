Welcome to the Magic Software web site! Quite likely you are here to access our free source code and documentation. The code base originated in 1991 to provide support for the technical news group comp.graphics.algorithms. Although the initial emphasis was on computer graphics, over the years the source code base has evolved to assist in problem solving in other fields including games programming, image analysis, and numerical methods.

Books. The source code at this site is free. Show your support by purchasing one of our books published by Morgan Kaufmann Publishers. The royalties are used by the company to produce more free source code. Two books currently available are 3D Game Engine Design, a book on building a graphics engine for games, and Geometric Tools for Computer Graphics, a book on useful topics that arise frequently in computer graphics. The next book is Game Physics and will be available near the end of 2003. This book is on the topic of physics and its implementation in a game development environment.

Services. Magic Software provides various services to the technical community including contract programming, on-site consulting, and training in our areas of expertise. An ongoing project we are involved with is algorithm development for the Medical Image Processing, Analysis, and Visualization package that is developed by researchers at the National Institutes of Health. Visit the services page for more information.

Source Code. The bulk of the content at this site is source code and documentation. We call the collective package the Wild Magic engine. The current version is 2.11 and was released on October 21, 2003. Please read the license agreement before downloading source code. Bug fixes and updates are posted to the update history page that is available at the source code page. Our source code is portable and runs on Microsoft Windows machines (9x/NT/2000/XP), Linux machines, and the Macintosh (OS 10.2.x and later). The graphics code contains a GLUT renderer that works on all platforms. On MS Windows we have renderers for Direct X and OpenGL (wgl). On the Macintosh we have OpenGL (agl). The graphics engine has both vertex shader and pixel shader support. Zip files containing the full distribution are available at the source code page; StuffIt files are available for the Macintosh.

Resources. The resources page has links to related web sites, a collection of interesting posts to the comp.graphics.algorithms news group, references to some papers, and a community page that contains links to downloadable software that users of our software have contributed.
