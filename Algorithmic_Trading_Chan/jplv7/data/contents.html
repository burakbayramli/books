-------- Demonstration data sets -------- 
All of these .dat files have an associated .txt file that describes the data

agriculture.dat : data on 24,473 zip codes land usage, animal inventories
agriculture.txt : Documentation

anselin.dat     : Anselin's Columbus OH 49 observation data set
wmat.dat        : 1st order standardized contiguity matrix in sparse format
anselin.txt     : Anselin.data Documentation
wmat.txt        : 1st order contiguity matrix Documentation

boston.dat      : Harrison and Rubinfeld Boston census tract data
boston.txt      : Documentation

crime.dat       : Ehrlich crime data
crime.txt       : Documentation

countyg.dat      : US county-level growth data set from C. H. Wheeler, J. Applied Econometrics
countyg.txt      : Documentation

elect.dat        : 1980 Presidential election 3,107 US counties, Pace and Barry, 1997
elect.txt        : Documentation

election.dat     : 1996 Presidential election 3,110 US counties, Clinton vs. Dole
election.txt     : Documentation

growthley.dat    : Fernandez, Ley and Steel, 72 countries and their growth rates
growthley.txt    : Documentation

grunfeld.dat     : Grunfeld investment data
grunfeld.txt     : Documentation

htoledo.dat      : Housing values 98 Census tracts Toledo, Ohio
htoledo.txt      : Documentation

house.dat        : data on 25,357 single family homes sold in Lucas County, Ohio, 1993-1998
house.txt        : Documentation

love.dat         : survey of MBA students feelings about being loved
love.txt         : Documentation

nelsonp.dat      : Nelson-Plosser time-series data
nelsonp.txt      : Documentation

ohioschool.dat   : data set on 1,965 Ohio Elementary School buildings for 2001-02 year
ohioschool.txt   : Documentation

school.dat       : 610 Ohio School districts proficiency tests
school.txt       : Documentation

spector.dat       : Spector and Mazzeo, 1980 probit data
spector.txt       : Documentation
